# Source Directory Structure

This directory contains the mobile app's client-side logic written in TypeScript.

## Folders

- **services/** - API clients, authentication, data sync services
- **models/** - TypeScript interfaces and data models
- **utils/** - Helper functions, validators, formatters
- **config/** - App configuration and constants
- **types/** - TypeScript type definitions
- **hooks/** - Custom React hooks

## Contributing

All contributions should follow the issues in the `backend/` folder at the root of the repository.
